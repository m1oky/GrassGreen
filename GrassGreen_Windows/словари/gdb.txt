user() = это ключ назначение
last_log() = это дата или время последнего захода 
net_tunnel_protocol() = тип сетевого туннеля 
net_baypass_state() = это статус подсистемы обхода ограничений (0 или 1)
net_gateway_route() = маршрут шлюза 
remember_me() = флаг постоянства сессии. Определяет, нужно ли сохранять и удерживать
 данные авторизации после закрытия процесса (0 или 1).
auth_token_hash() = задел на будущее под зашифрованный
 сессионный ключ или хэш, чтобы никто не мог угнать сессию.
env_target_runtime() = целевая среда запуска (например, fabric-1.20.1, terraria-v1.4.4)
env_exec_flags() = дополнительные флаги оптимизации при запуске 
процесса (туда можно будет автоматом закидывать аргументы выделения оперативной памяти для Sodium/Lithium)
net_bypass_config() = путь к файлу конфигурации прокси/туннеля (например, vless_reality.json)
net_ping_ms() = задержка до игрового сервера или узла туннеля в миллисекундах (для вывода в UI)
env_allocated_ram() = количество выделяемой оперативной памяти в мегабайтах (например, 4096)
env_java_path() = кастомный путь к исполняемому файлу Java (javaw.exe) для запуска майна
sys_fps_stabilizer() = флаг включения внутренней оптимизации процессов Windows под Forge/Fabric (0 или 1)
sys_gpu_force_integrated() = принудительный запуск на встроенной графике Intel UHD при сбое дискретной карты (0 или 1)
