# GrassGreen 🌿

Это мой кастомный язык программирования, написанный на C++. 
Проект содержит рабочие библиотеки для парсинга, кастомных вычислений и математической логики.

## Особенности проекта
* **Стабильная версия:** Оригинальный релиз, полностью оптимизированный под Windows.
* **CoreMath & FastMath:** Собственные быстрые алгоритмы вычислений и кастомный квантовый регистр для хранения состояний кубитов.
* **Скрипты:** Поддержка выполнения файлов с расширением `.grass`.
